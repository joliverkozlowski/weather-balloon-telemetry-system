# Weather Balloon Telemetry System

Undergraduate thesis developed at the Instituto Tecnológico de Aeronáutica (ITA) in 2022.

This project presents the design, implementation, and testing of a low-cost telemetry system intended for scientific payloads carried by stratospheric weather balloons.

The developed system provides:

- Telemetry communication between payload and ground station
- GPS position tracking
- Electrical power distribution
- Thermal protection for payloads
- Support for generic scientific experiments

## Technologies

- STM32F103 (Blue Pill)
- LoRa E220-900T22D
- GPS NEO-6M
- KiCad
- Custom PCB
- Custom 915 MHz antenna

## Project Highlights

- Complete system engineering process
- Requirements definition
- Hardware design
- PCB development
- Antenna design and characterization
- Link budget analysis
- Prototype manufacturing
- Functional testing
